<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

-   Node.js

### Installation

To get a local copy up and running with:

#### Node.js

1. Clone the repo

    ```sh
    git clone https://github.com/gspalato/ecobucks.git
    ```

2. Install dependencies

    ```sh
    yarn
    ```

3. Start it
    ```sh
    yarn start
    ```
